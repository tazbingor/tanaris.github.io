Django Web开发01:项目规划及开发环境配置

发表于 2018-04-08

目标是以Django为基础开发一个视频网站

网站设计思路

内容

开发视频网站,其主要的内容就是视频在页面中的展现,所以这个网站需要一个首页(屁话).为了方便用户选择视频内容,还需要一个分类的页面.顺着一般网站开发的思路,肯定也需要一个用户的登录注册的页面.继续往下捋一捋思路,肯定会有更多的需求提出来.此时,需要抽丝剥茧,将网站最核心的功能一一列出.也是为了能让网站先跑起来

  1. 首页
  2. 视频分类页面
  3. 视频详情(主视频播放页面)
    1. 视频播放
    2. 用户评论
  4. 用户认证页面
    1. 注册
    2. 登录
  5. 观看历史页面

实现功能

依照上述给出的页面内容,逐步丰满网站的功能,主要的功能有一下两条

####视频

  1. 视频分类,基本的上下页翻页的功能
  2. 视频评论,在视频详情页面下发表评论
  3. 视频分享

####用户

  1. 注册
  2. 登录
